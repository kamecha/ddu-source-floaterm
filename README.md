# ddu-source-floaterm

floaterm source for ddu.vim

# Required

## denops.vim

https://github.com/vim-denops/denops.vim

## ddu.vim

https://github.com/Shougo/ddu.vim

## vim-floaterm

https://github.com/voldikss/vim-floaterm

# Example

```
call ddu#start(#{ sources: [#{ name: 'floaterm' }] })
```
